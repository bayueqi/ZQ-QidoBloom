# 琪窦初开 (QidoBloom)

## 项目介绍

琪窦初开（QidoBloom）是一款功能强大的浏览器扩展，为您提供美观、高效的个人导航主页。它不仅是一个简单的书签集合，更是一个全方位的上网入口，集成了快捷访问、搜索引擎管理、个性化分组、壁纸更换等多种功能，让您的网络冲浪体验焕然一新。

### 核心特性

- 🎨 **美观界面**：现代化设计，支持自定义壁纸
- 🔍 **多搜索引擎**：集成主流搜索引擎，支持自定义添加
- 📁 **网站分组**：可创建多个分组，分类管理常用网站
- 🌐 **快捷访问**：一键直达常用网站，提高工作效率
- 🎯 **智能搜索**：快速切换搜索引擎，支持键盘快捷键
- 💾 **本地存储**：数据保存在本地，保护隐私安全
- 🖼️ **壁纸管理**：支持自定义壁纸，打造个性化界面
- 📤 **数据导入导出**：支持数据备份和迁移，确保数据安全
- 🚀 **浏览器扩展**：作为浏览器扩展运行，随时访问

## 快速开始

### 安装部署

#### 方法一：作为浏览器扩展安装（推荐）

1. **准备扩展**
   - 下载并解压本项目到本地
   - 确保项目目录结构完整

2. **安装到浏览器**
   - **Chrome / Edge**
     1. 打开浏览器，访问 `chrome://extensions` 或 `edge://extensions`
     2. 开启「开发者模式」
     3. 点击「加载已解压的扩展程序」
     4. 选择项目根目录
     5. 扩展安装完成，可在扩展栏中固定

   - **Firefox**
     1. 打开浏览器，访问 `about:debugging#/runtime/this-firefox`
     2. 点击「临时载入附加组件」
     3. 选择项目中的 `manifest.json` 文件（如果没有，需要创建）
     4. 扩展临时安装完成

3. **访问扩展**
   - 点击浏览器扩展栏中的琪窦初开图标
   - 或在新标签页中设置为默认主页

#### 方法二：本地文件方式

1. **下载项目**
   - 克隆或下载本项目到本地
   - 解压到任意目录

2. **启动服务**
   - 方法一：使用本地服务器
     ```bash
     # 使用Python启动简易服务器
     python -m http.server 8000
     
     # 或使用Node.js（如果已安装）
     npx http-server -p 8000
     ```
   - 方法二：直接打开文件
     - 双击 `index.html` 文件在浏览器中打开

3. **访问应用**
   - 在浏览器中访问：`http://localhost:8000`（如果使用本地服务器）
   - 或直接打开 `index.html` 文件

## 功能使用

### 添加常用网站

1. 点击页面顶部的「管理」按钮
2. 选择「分组管理」
3. 创建新分组或选择现有分组
4. 点击「添加网站」按钮
5. 填写网站名称、URL，系统会自动获取网站图标
6. 点击「保存」按钮完成添加

### 管理搜索引擎

1. 点击页面顶部的「管理」按钮
2. 选择「搜索引擎管理」
3. 在这里可以：
   - 添加新的搜索引擎
   - 编辑现有搜索引擎
   - 删除不需要的搜索引擎
   - 设置默认搜索引擎

### 自定义壁纸

1. 点击页面顶部的「管理」按钮
2. 选择「壁纸管理」
3. 在这里可以：
   - 选择内置壁纸
   - 上传自定义壁纸（支持图片文件）
   - 恢复默认白色背景
   - 实时预览壁纸效果

### 数据导入导出

1. 点击页面顶部的「管理」按钮
2. 选择「数据管理」
3. 在这里可以：
   - **导出数据**：将所有配置导出为JSON文件，用于备份
   - **导入数据**：从JSON文件恢复之前的配置
   - **重置数据**：清空所有数据，恢复到初始状态

### 浏览器扩展设置

1. **固定扩展**：在浏览器扩展栏中右键点击琪窦初开图标，选择「固定」
2. **设置为新标签页**：在扩展设置中，可以将琪窦初开设置为默认新标签页
3. **快捷键设置**：在浏览器扩展管理中，可为琪窦初开设置自定义快捷键

## 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **存储方案**：localStorage（本地存储）
- **图标服务**：[favicon.im](https://favicon.im)（网站图标）
- **字体**：[Google Fonts](https://fonts.google.com)（Inter字体）
